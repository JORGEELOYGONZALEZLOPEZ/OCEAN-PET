MASCOTA OCÉANADocumentación de Proyecto Full-Stack.Jorge Eloy Gonzalez Lopez - 5C Desarrollo de Software Multiplataforma.🚀 Proyecto OCEAN-PET + ViteEste es un proyecto Full-Stack moderno diseñado para gestionar la recuperación de plástico en el Océano Pacífico y el Golfo de México, transformando materiales recuperados en productos tecnológicos y sostenibles.📋 DescripciónEste repositorio funciona como una plataforma integral que conecta la recolección de residuos marinos con una interfaz de comercio electrónico. Cubre desde la gestión de inventario en base de datos hasta la navegación fluida de un catálogo dinámico.🛠️ Stack TecnológicoEl proyecto utiliza un ecosistema Full-Stack JavaScript de alto rendimiento:Frontend: React.js con arquitectura de componentes funcionales y Hooks para una experiencia de usuario fluida.Backend: Node.js y Express.js para la construcción de una API REST robusta.Base de Datos: MySQL gestionado mediante el ORM Sequelize, facilitando la administración de modelos y relaciones.Seguridad: dotenv para la gestión de variables de entorno y protección de llaves sensibles.📊 Estructura y Lógica de NegocioEl sistema implementa una arquitectura modular para separar la lógica de datos de la interfaz:Base de Datos (Modelos)EntidadPropósitoProductosGestión de inventario reciclado: nombre, precio, stock y descripción técnica.UsuariosControl de perfiles para clientes y administración de roles.Carrito / ÓrdenesGestión de transacciones y vinculación de pedidos antes de la compra.💻 Arquitectura de InterfazLa interfaz se basa en la reutilización de componentes para un mantenimiento eficiente:Nuestro Catálogo Tecnológico: Módulo dinámico que consume la API del backend para mostrar productos en tiempo real.Gestión de Carrito: Lógica avanzada para modificar cantidades, eliminar artículos y calcular totales dinámicamente.Vistas Dinámicas: Navegación Single-Page Application (SPA) mediante la función cambiarVista.🌐 Integraciones y SeguridadOpenWeather API: Implementación de datos climáticos para monitorear condiciones de recolección en tiempo real.GitHub Push Protection: Reglas activas de seguridad para prevenir la exposición de llaves API.Control de Versiones: Uso estricto de .gitignore para proteger archivos de configuración.🛠️ Requisitos PreviosAntes de comenzar, asegúrate de tener instalado Node.js y npm en tu sistema. Puedes verificarlo con los siguientes comandos:Node.js: node -vnpm: npm -v🚀 Instalación y UsoClona el repositorio:git clone https://github.com/JORGEELOYGONZALEZLOPEZ/OCEAN-PET.gitInstala dependencias:npm installConfigura tu .env:Añade tus credenciales de MySQL y tu clave de OpenWeather.Inicia el proyecto:npm start
